# UX 코치 세션

### 혼란스럽고 여러 의미를 가진 아이콘 지양
- 과거의 아이콘 지양
- 독특한 아이콘 지양

### 레이블을 통한 사용성 추구
- 직관적인 아이콘에 레이블을 붙이면 사용자의 80펴센트 이상이 예측
- 레이블이 없으면 60%
- 레이블이 없고 독특한 아이콘이면 40% 이하의 인식
